# 🔐 Gerador de Senhas em Python

Este projeto é um **gerador de senhas seguras** desenvolvido em Python.  
O objetivo foi praticar lógica de programação e uso da biblioteca `random`.

A senha gerada segue o seguinte padrão:
- 2 letras **minúsculas**
- 2 letras **maiúsculas**
- 2 **números**
- 2 **sinais de pontuação**

Exemplo de saída:
```
aB9!cD3@
```

---

## 🚀 Funcionalidades
- Geração automática de senhas fortes.
- Uso de diferentes tipos de caracteres.
- Simples e rápido de executar.

---

## 📦 Tecnologias usadas
- **Python 3**
- Biblioteca padrão `random` (sem dependências externas)

---

## ▶️ Como executar
1. Clone este repositório:
   ```bash
   git clone https://github.com/NBittent/Random-Password-Generator
   ```
2. Entre na pasta do projeto:
   ```bash
   cd Random-Password-Generator
   ```
3. Execute o script:
   ```bash
   python main.py
   ```
